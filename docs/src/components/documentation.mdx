## Usage

```js
import {hexToHSL, hslToHex} from '@barelyhuman/tocolor'

function darker(percentage, hex) {
	if (!percentage) {
		return this
	}
	const {h, s, l} = hexToHSL(hex)
	const _afterDarken = l - percentage
	const value = hslToHex(h, s, _afterDarken)
	return value
}
```

## API

The library comes bundled with the following functions, each one is typesafe and have the same base response.

1. `toHex` calls return a string value of hex without the starting `#`
2. `toRGB` return an object of the form {r,g,b}
3. `toHSL` return an object of the form {h,s,l}
4. `rgb` functions take in `(r,g,b)`
5. `hsl` functions take in `(h,s,l)`
6. `hex` functions take in a valid hex string `eg:(#fff,#ffffff)`

### Exported Functions

- `rgbToHex(r,g,b)`
- `hslToHex(h,s,l)`
- `hslToRGB(h,s,l)`
- `rgbToHSL(r,g,b)`
- `hexToRGB(hexString)`
- `hexToHSL(hexString)`
- `parseToHex(anyColorString) => anyColorString => rgb(x,y,z) | hsl(x,y,z) | hexString`
